RYSE & DeFND

Developed on Unity: C# for the backedn & Blender for 3D design

In this video we can see the game starting with a finite set of currency and the two turrets the user can use. When a turret is selected it will display a green grid if the user has enough money to place the given turret. If the user does not have enough money the grid will be red. When the user clicks on a turret they will have the option to upgrade if they have enough money or sell the given turret. Waves will continue to come with small window of time untill the level is complete. There are mutiple levels for the user to choose from aswell.

Difficulties: A struggling point was when I needed to get each turret to point at the target in which they were going to fire. Another big challange actually creating the 3D design of the wizard with Blender since it was my first 3D design. I also struggled when doing the small anamations when something gets destroyed and destroying things at the right time.

How I Overcame: I was able to use lessons learned in Linear Algebra and use matrix multiplication and manipulation to get them pointing the right way. I able to complete the 3D design through 12 hours of non-stop trial and error, trying new things and removing things that didn't. The last struggle was overcame by understanding how Unity calls their Update() function and whether it was done upon impact or very close to the impact of the bullets.
