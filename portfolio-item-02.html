___________________ Developed with Xamarin Forms: using C#, XAML, and Google Firebase

In this video you will be able to see a user creating an account, ensuring he enters the same password. You will also notice that everything gets updated to the Firebase database in realtime when the user creates their account. Once the user logs in they will be able to View Lots, Search by tieme, Thier Reservations, and Log out. In the video the user will View Lots and reserve a space for 2 hours. The user will then be able to see his reservation history. Then when the user return to the Lot he will notice that his space is no longer there and it will show that 1 space is Busy. Then the user logs out.

Difficulties: The greatest challange to this was being able to set up communication with Firebase database as it was my first time using a realtime database instead of a local one. Another struggle was being able to launch the app and make it into a Cross Platform Mobile Application.

How I Overcome: It took me about a week after reading countless articles online and those published by Google but, after trying several different methods I was able to get it up and running. Being able to get the application to be Cross Platform was simplier than imagined as Xamarin Forms took care of most of the heavy lifting.
